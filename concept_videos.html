For Dr. Rennie: how to attach videos

Each video card has a videoUrl field in the JavaScript config (top of the script block). Edit the file directly to attach videos. Both Canvas Studio and YouTube embed URLs work in the same field.

  1. Canvas Studio (your originals): open the video, click Share, then Embed. Copy the URL from the iframe's src attribute (looks like https://yourschool.instructuremedia.com/embed/abc123).
  2. YouTube (curated): open the video on YouTube, click Share, then Embed. Copy the URL from the iframe's src attribute (looks like https://www.youtube.com/embed/VIDEO_ID).
  3. Paste the URL into the matching lecture's videoUrl field. Save and re-deploy.

The status pill on each card auto-detects what kind of URL you pasted and shows it accordingly.
