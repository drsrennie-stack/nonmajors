How this workbook works

Print, handwrite, scan, submit. Complete every section. Drawings, labeling, tables, and clinical reasoning all earn points.

About the drawings in this workbook. For each drawing, find a clear reference in OpenStax Anatomy and Physiology 2e or any anatomy atlas. You may either (a) draw it by hand in the workbook box, or (b) build it digitally in Google Docs, Google Drawings, Word, or PowerPoint using shapes, lines, and text labels, then attach the file to your Canvas submission.

If you build digitally, you must submit revision history showing you built the diagram in stages over time. In Google Docs and Google Drawings: File → Version history → See version history. A flat image with no revision history does not meet this requirement.

Either way, you must build it yourself, structure by structure. Copying a finished image (from Google Images, an AI tool, or a peer) earns zero on the assignment and is reported as academic dishonesty. Submission to Canvas certifies the work as your own.

Anatomy · Drawing from Reference

Respiratory membrane and nephron with three transport processes

Find each diagram in OpenStax Anatomy and Physiology 2e or any anatomy atlas. Reproduce each diagram in the box and label every required component.

Drawing 1: Respiratory membrane in cross-section

Draw a single alveolus next to a capillary, showing the path that oxygen takes from the alveolar lumen into a red blood cell. Required components: alveolar lumen with oxygen, type I pneumocyte, alveolar basement membrane, fused basement membrane (interstitial space), capillary endothelium, capillary lumen with red blood cell, oxygen molecules with arrows showing the diffusion path. Also include one type II pneumocyte and label what it secretes.

Drawing space

Drawing 2: Nephron with the three urine-forming processes
